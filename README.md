# Image-Based-Recognition
## Project Title

Data Preprocessing for COVID-19 Dataset Using Python

## Project Description
This project focuses on preparing a COVID-19 dataset for machine learning by applying essential data preprocessing techniques. The dataset is cleaned and transformed to improve its quality before model training. Missing values are handled using mean imputation, categorical features are converted into numerical values using One-Hot Encoding and Label Encoding, and the dataset is divided into training and testing sets. Finally, feature scaling is applied to normalize numerical data, ensuring better performance of machine learning algorithms.

The project is implemented using Python, Pandas, NumPy, and Scikit-learn, demonstrating the complete preprocessing workflow required before building predictive machine learning models.

## Key Features
*Imported and analyzed the COVID-19 dataset.

*Handled missing values using SimpleImputer.
Encoded categorical variables using One-Hot Encoder.
Converted target labels using Label Encoder.
Split the dataset into 80% training and 20% testing data.
Applied StandardScaler for feature normalization.
Prepared clean and standardized data for machine learning.

Successfully transformed the raw COVID-19 dataset into a clean, structured, and machine learning-ready dataset by performing all essential preprocessing steps.
